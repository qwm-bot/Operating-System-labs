# lab1:最小可执行内核
## 实验目的
- 使用`链接脚本`描述内存布局；
- 进行`交叉编译`沈城可执行文件，进而生成内核镜像；
- 使用OpenSBI作为bootloader加载内核镜像，并使用Qemu进行模拟；
- 使用OpenSBI提供的服务，在屏幕上格式化打印字符串用于以后调试。
## 实验内容
